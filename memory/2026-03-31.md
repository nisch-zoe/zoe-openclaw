## Research Pipelines — Built and Running

Set up hybrid research pipelines for competition analysis and content strategy:

- **competitor-pulse**: tracks Axio, Fold, Money Manager, Spendrix via App Store (India) metadata + Reddit mentions + HN threads. Shows ratings, versions, update dates, and user sentiment.
- **content-trends**: scans Reddit hot posts, HN top stories, and dev.to trending articles. Scores them against content pillars (iOS eng, privacy/fintech, indie maker, tools, observations). Produces a pillar heat map.
- **reddit-listen**: monitors target subreddits (personalfinanceindia, india, iphone, developersIndia) for keywords. Classifies posts by intent: pain points, feature requests, content opportunities. Flags competitor mentions.

All pipelines use free, no-auth APIs. Output goes to `knowledge/reference/research/` as dated Markdown reports with raw JSON snapshots. Each report includes an LLM analysis prompt section for synthesis.

### Architecture
Hybrid model: deterministic layer (scoring, keyword matching, intent classification) handles structured data collection and ranking. LLM layer (prompt templates in each report) handles strategic synthesis, pattern recognition, and content angle generation. The agent can read the reports and synthesize on demand, or the prompts can be fed to any LLM.

Script: `node workspace/scripts/research.js {competitor-pulse|content-trends|reddit-listen|full-scan}`
Config: `knowledge/reference/research/config.json`

## Social Media Trend API Research

Researched third-party APIs for monitoring trending content and top-performing posts across Twitter/X, Instagram, LinkedIn (platforms not covered by our existing free pipelines).

**Top picks:**
- **API Direct** (apidirect.io) — unified API, $0.003-0.008/req, covers all target platforms, pay-as-you-go with free tier. Best fit for extending research.js.
- **SociaVault** (sociavault.com) — credit-based ($29 for 6k credits), 25+ platforms, good for competitor content analysis.
- **KeyAPI** (keyapi.ai) — broadest coverage (20+ platforms incl. LinkedIn), AI-agent-first design, historical data up to 1000 days.
- **ContentStats.io** — $0.015/snapshot for hourly tracking of specific posts over time (good for format performance analysis later).

Full research saved to `knowledge/reference/research/social-trend-apis-research.md`.
